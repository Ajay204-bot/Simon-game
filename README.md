Simon Game
A Memory Challenge Game Built with JavaScript
      Overview
            A simple, fun, memory-based Simon Game built using HTML, CSS, JavaScript, and jQuery.
            This project generates random sequences of colored buttons that the user must repeat in the same order.
            Each successful round increases the level and score.

      Features

            1.Random color sequence generation

            2.Increasing difficulty (faster blinking per level)

            3.Score + High Score tracking

            4.Reset High Score button

            5.Sound and visual effects

            6.Game Over detection & restart

            7.Responsive and smooth UI for laptop

      How to Play
            1.Press any key to start.

            2.Watch the blinking color sequence.

            3.Click the buttons in the exact same order.

            4.Complete the pattern → level increases & score increases.
            
            5.Make a mistake → Game Over → Press any key to restart.

            6.Click Reset High Score to set High Score back to 0.

      Tech Stack
            HTML5
            CSS3 (animations + gradients)
            JavaScript
            jQuery
